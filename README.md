# Bonobo branding
This repository contains the images for Bonobo's branding, as well as licensing information.

## Square icons
Bonobo's square icons can be found under the `square/` directory in 128x128, 48x48, 32x32, and 16x16 resolutions.

## Rectangular wordmark
Bonobo's rectangular wordmark can be found under the `wordmark/` directory.
